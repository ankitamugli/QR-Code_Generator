# QR Code Generator

## 📌 Project Overview

QR Code Generator is a simple web application that allows users to generate QR codes instantly from text, URLs, or other input data. The application provides a clean and responsive interface, making QR code creation quick and easy.

## 🚀 Features

- Generate QR codes instantly
- Convert text and URLs into QR codes
- Responsive and user-friendly design
- Fast and lightweight application
- Easy to use interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript


## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/ankitamugli/QR-Code_Generator.git
```

2. Open the project folder.

3. Run the application by opening `index.html` in your browser.

## 💻 Usage

1. Enter a URL or text in the input field.
2. Click the **Generate QR Code** button.
3. View the generated QR code.
4. Scan the QR code using any QR scanner.

## 📸 Output

The application generates a QR code based on the user input, which can be scanned using a smartphone or QR scanner.
